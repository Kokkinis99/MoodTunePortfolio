MoodTune

Learning Angular and Ionic by building something real. Log how you feel — MoodTune picks a Spotify playlist to match. It never shipped, but everything on this page works.

Calendar

Tap to expand. Click any date to change the mood. The colour of the whole calendar follows the selected day's mood.

Save your streak

Tap the card 7 times to save your MoodWeek streak. Each tap fires 30 CSS particles — cos()/sin() radial dispersion, no canvas.

What I learned

Spotify PKCE OAuth

Auth-code-with-PKCE — a code verifier generated client-side, SHA-256 hashed, exchanged for an access token. No backend secret, no server needed.

Firebase + Firestore

First time setting up a NoSQL database from scratch — mood history, streak data, user profiles. Also the first time setting up a Firebase project.

Spotify Web API

Track search, playlist creation, auth scopes. Plus the Web Playback SDK so playback stays inside the app — no redirecting out to Spotify.

ReccoBeats

A recommendation engine scored on mood–audio feature correlations. First time consuming an API I didn't write and had to properly understand.

Angular + Ionic patterns

Standalone components, OnPush change detection, NgZone, CDK Drag-Drop — most of this was new territory when the project started.

What I found interesting

Spring animations

The calendar expand/collapse uses a linear() spring curve — controlled overshoot that makes the motion feel physical rather than mechanical. Getting it right took longer than the rest of the calendar.

Mood theming

One CSS variable on :root. The whole app recolours in a single frame — every surface, every accent, every gradient responds immediately.

CSS particles

cos()/sin() for radial dispersion, per-particle custom properties for angle and distance. 30 particles per burst, no canvas, no library.

MoodWeek streaks

Miss a day — the broken streak modal fires. Tap the card 7 times to save it, each tap revealing more of the mood image underneath the grayscale overlay.

FLIP animations

Polaroid expand uses reverse-FLIP — measure the final rect, apply a transform to jump to the start position, then clear it so the browser interpolates back.
